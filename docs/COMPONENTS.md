# UI Component Directory

A guide to reusable components in the `3d-ai-maker.tech` design system.

## Layout & Common UI
- `Header.tsx`: Navigation and User Profile.
- `BusinessModal.tsx`: Service/Price display modal.
- `AdminDashboard.tsx`: Base layout for management functions.

## 3D Tools (Ready for 3D Printing)
- `QRCodeTool.tsx`: Generates 3D-printable QR codes with STL export.
- `LithophaneTool.tsx`: (In Development) Image to 3D light-box conversion.

## Animations (Lottie-like Visuals)
- `SyntheticCreationAnimation.tsx`: Purple-themed SVG hardware animation (Purple circuit traces).
- `PhysicalManifestationAnimation.tsx`: Blue-themed hardware manufacturing animation.

## Interactive Data
- `CodeTree.tsx`: Hierarchical organization view (User account management tree).
- `DashboardCodeTree.tsx`: Simplified account management tree for standard users.

## Styling Tokens (Tailwind)
- `glass-panel`: Standard frosted glass container.
- `glass-panel-deep`: Darker, more prominent panel for emphasis.
- `text-primary`: Primary blue color (`#38bdf8`).
- `text-secondary`: Purple accent color.
