# 3D 打印轻量化工具原理课 (S1-S10) 规划方案

本课程体系旨在补足传统 3D 打印教育中“只打不学”、“等待时间长”的痛点，通过线上即时生成的算法工具，让学生在打印前深度参与原理设计。

## 课程列表及算法原理规划

| 课号 | 课程名称 | 核心工具 | 算法原理 (Algorithmic Principles) | 教学互动重点 |
| :--- | :--- | :--- | :--- | :--- |
| **S1** | **光影魔术师** | Lithophane 生成器 | **灰度映射算法**：将图像像素的灰度值线性映射为 3D 模型的厚度，利用材料透光性重建图像。 | 【可预览】照片转浮雕，理解透光率与厚度的关系。 |
| **S2** | **物理链接器** | 3D 二维码工具 | **高低差投影算法**：利用黑白矩阵生成具有高度落差的立方体阵列，通过光影阴影形成可扫描特征。 | 制作个人数字名片，理解纠错码（ECC）与物理识别。 |
| **S3** | **柔性折叠结构** | 活铰链 (Hinge) | **应力分散算法**：计算超薄结构在弯折时的形变率，生成可一次性打印的免装配机械结构。 | 打印“不用螺丝的折叠盒”，理解材料韧性与几何结构。 |
| **S4** | **重力平衡大师** | 不倒翁平衡工具 | **质心偏移算法**：通过调整模型内部填充率分布或几何偏心，计算物体在不同姿态下的重心平衡点。 | 物理实验：预测并验证模型的平衡状态。 |
| **S5** | **拓扑格子构造** | 点阵 (Lattice) | **空间平铺算法**：使用最小表面积函数（TPMS）生成高强度、极轻量的蜂窝状内部支撑结构。 | 体验“减重不减效”，学习现代工业设计中的轻量化。 |
| **S6** | **曼陀罗算法艺术** | 生成式几何工具 | **对称递归算法**：基于极坐标系下的数学函数，通过调整参数生成具有复杂镜像对称性的装饰模型。 | 参数化设计审美，每个人生成的图案都独一无二。 |
| **S7** | **斐波那契螺旋** | 自然频率工具 | **黄金分割算法**：根据 1.618 比例生成螺旋排列的叶片或凹槽，模拟植物生长的空间最优化分布。 | 跨学科结合：数学、生物与 3D 打印的耦合。 |
| **S8** | **Voronoi 细胞骨架** | 泰森多边形工具 | **空间分区算法**：在空间中随机撒点，计算每个点到周围区域的垂线平分线，形成有机细胞化外壳。 | 给已有模型穿上“生物外衣”，理解自然界的结构效率。 |
| **S9** | **声波纹理解码** | 音频视觉化工具 | **傅里叶变换 (FFT)**：将声音频率转换为振幅序列，映射为 3D 模型的起伏波纹或环绕纹理。 | 打印“声音的形状”，理解波动原理与数字化记录。 |
| **S10** | **AI 创客助手** | Prompt-to-3D | **大模型向量生成**：将语义描述转化为 3D 参数脚本，实现从文字直达三维实体的端到端快速生成。 | 未来趋势：AI 赋能创客，实现从想法到实体的零操作跨越。 |

## 访问权限逻辑
- **S1 (光影魔术师)**：作为体验课，允许未登录用户访问工具页与原理分析，作为引导流量入口。
- **S2-S10**：所有交互交互工具及其对应的原理教学页均受 **“登录墙”** 限制，必须由通过 B2B 激活码授权的账号访问。
