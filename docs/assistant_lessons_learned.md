# 🤖 AI 助手易错点与项目知识库

本文档记录了 AI 助手在开发“3D打印AI创客”项目过程中常见的理解偏差或错误，以便后续保持高度准确。

## 0. 核心沟通与操作规范
- **禁止私自调用浏览器验证**: 不要在未确认交互逻辑完成前私自启动 `browser_subagent` 跑测试。
- **遇到问题及时沟通**: 开发工具或遇到逻辑阻塞时，应先提供建议与用户讨论，而不是自行盲目决定方案。
- **文化属性定位修正（如：马勺脸谱）**: 在设计非遗（L1）课程时，需准确把握文化内涵。例如“关中马勺”，它的本质是古代民间祈福辟邪的“手办/图腾（类似今天的二次元谷子）”，其核心公式是“在特定物件（马勺）上装饰精神图腾”。因此，互动设计应围绕**图腾元素的组合与装饰**，而不是单纯的“画脸/五官拼接”（后者应留给“秦腔脸谱”等具有明确戏剧脸谱特征的课程）。


## 1. 课程体系混淆 (核心错误点)
- **L1 系列 (16节非遗创客课)**: 
    - **定位**: 使用 **3D 打印笔**结合“非遗文化”的入门课程。
    - **代表**: 景泰蓝 (`cloisonne`)、**关中马勺 (`mashao-mask`)**、宋锦、皮影。
    - **逻辑**: 文化溯源 -> 数字化交互 (如对称画板) -> 3D 打印笔实物制作。
- **S1-S10 系列 (10节原理小工具课)**: 
    - **定位**: 侧重 **3D 生成算法与几何原理**的高阶工具。
    - **代表**: 光影魔术师 (S1, Lithophane)、3D 二维码 (S2)、斐波那契螺旋。
    - **逻辑**: 算法原理讲解 -> 参数化工具生成 -> 3D 打印导出。

### 🚩 历史错误记录
- **误区**: 曾将“关中马勺”误认为 S 系列的算法工具，或在查找文件时定位到不相干的展示 json。
- **更正**: 确认“关中马勺”属于 L1 系列的第七课 (L1-07)，其页面位于 `src/app/course/l1/mashao-mask/page.tsx`。

## 2. 文件路径规范
- **课程详情页**: `src/app/course/l1/[slug]/page.tsx`
- **算法工具页**: `src/app/tools/[id]/page.tsx`
- **课件数据**: `public/presentations/[slug]/refined.json`

## 3. UI 风格演变
- 正在从极客深色 (`slate-900`) 向 B2B 专业亮色/毛玻璃风格转型。
- 在构建 L1 课程页时，应参考 `cloisonne/page.tsx` 的布局：采用大圆角卡片 (`rounded-[48px]`) 和沉浸式动画切换。
