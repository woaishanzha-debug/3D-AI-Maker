# 3D AI Maker 项目叙事逻辑与系统架构全案 (V1.0)

本文件旨在事无巨细地梳理 “3D AI Maker” 平台的叙事逻辑、功能架构、技术栈及未来演进规划，为中长期的内容填充、功能优化及品牌升级提供核心基准。

---

## 一、 品牌核心叙事逻辑 (Narrative Logic)

### 1.1 核心主张：从 “数字智能” 到 “物理触达”
平台通过 **AI (人工智能)** 与 **3D Printing (三维实体制造)** 的物理耦合，打破传统创客教育中 “建模难” 与 “创意枯竭” 的瓶颈。
*   **叙事起点**：AI 原理启蒙与提示词工程（解决 “想什么” 和 “怎么画”）。
*   **叙事中转**：参数化 3D 工具（S1-S10）（解决 “如何通过算法生成复杂的实体结构”）。
*   **叙事终点**：AI 创客综合系统（L1-L3）（解决 “如何用 AI 辅助设计并制造出具有文化底蕴或功能性的实物”）。

### 1.2 用户旅程 (User Journey)
1.  **感知 (Awareness)**：首页展示 STEAM 雷达图与精美学生作品，建立 “高科技、高美感” 的第一印象。
2.  **探索 (Exploration)**：通过 3D 打印原则课 (S1-S10) 的预览（如 Lithophane），体验算法生成实体的快感。
3.  **沉浸 (Immersion)**：注册登录后，锁定更具深度的 AI 创客课程 (L1/L2) 与智能编程套件 (L3)。
4.  **成就 (Achievement)**：生成 STL 模型 -> 物理打印 -> 拍照上传至学生作品集 -> 参加比赛。

## 二、 信息架构 (Information Architecture)

### 2.1 导航体系 (Navigation Cluster)
*   **AI 教育课程**：

    *   学生作品集（Gallery/Showcase）。
    *   比赛通道（Competition Portal）。

## 三、 功能模块深度拆解 (Functional Modules)

### 3.1 首页核心组件：STEAM 进化雷达图 (STEAM Radar)
*   **逻辑**：可视化展示课程对学生“科学、技术、工程、艺术、数学”五大维度的能力提升。
*   **技术**：基于 `Recharts` + `framer-motion` 实现动态加载与防 SSR 水合偏差。

### 3.2 课程/工具墙：分阶授权逻辑
*   **预览模式 (S1)**：无需登录，展示完整工具能力（Lithophane 浮雕像生成）。
*   **登录墙逻辑 (S2-S10)**：强制引导至 `/login` 或 `/register`，通过点击触发锁定动画，建立“高价值内容”的心理预期。
*   **UI 映射**：Red Lock 图标 + 激活码授权文案。

### 3.3 核心工具：3D 二维码生成器 (QRCodeTool)
*   **叙事逻辑**：展示数字信息（URL）如何转化为物理拓扑结构（点阵高度）。
*   **技术能力**：实时 3D 预览 (`Three.js`)、参数控制（厚度、边框、容错率）、STL 导出。

### 3.4 响应式系统 (Responsive Header)
*   **移动端适配**：
    *   多级折叠手风琴 (Accordion) 菜单。
    *   暗色 Overlay 遮挡，解决文字视觉冲突。
    *   汉堡菜单动画与同步的文案标签。

## 四、 技术架构 (Technical Stack)

| 维度 | 技术选型 | 备注 |
| :--- | :--- | :--- |
| **前端框架** | Next.js 14 (App Router) | 强 SEO 友好，支持 React Server Components |
| **样式方案** | Tailwind CSS + Vanilla CSS | 玻璃拟态 (Glassmorphism)、渐变色设计 |
| **数据库** | Prisma + PostgreSQL (Vercel Postgres) | 管理用户、课程进度、工具日志 |
| **身份认证** | NextAuth.js | 支持邮箱、OAuth 等多重登录方式 |
| **3D 引擎** | Three.js | 用于工具页面的实时渲染与预览 |
| **动画效果** | Framer Motion | 提升 UX 质感，实现转场与悬停反馈 |
| **构建工具** | PostCSS + Autoprefixer | 确保移动端浏览器样式兼容性 |

## 六、 中长期规划 (Long-term Vision)

### 阶段 1：AI 创作 Agent 深度集成 (1-3月)
*   **功能**：在网站内集成 “Prompt-to-3D” 实验室。
*   **目标**：用户输入一段话，AI 自动生成 3D 模型代码（基于 OpenSCAD 或 Three.js 脚本）。

### 阶段 2：在线 3D 学习管理系统 (LMS) 升级 (3-6月)
*   **功能**：用户进度追踪、作业上传、导师在线批改。
*   **目标**：从单向的展示平台转变为双向互动的智慧教室。

### 阶段 3：非遗数字化资产库 (6-12月)
*   **功能**：建立一个可 3D 打印的中国传统纹样资产库。
*   **目标**：将中华优秀传统文化通过 3D 打印笔与 AI 技术进行普惠化传播。
